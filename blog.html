Leap Years (3/1)

Yeasterday was a leap day. Leap Years are kind of weired so I thought I would write a little bit about them here.

We consider a normal year to be 35 days long. BUT it takes the Earth 365 days, 6 hours, 9 minutes, and 10 seconds to orbit the Sun. If we did not fin a way to account for that extra time the seasons would sowly become out of sync with the calander. For example, over the course of only 720 years, christmas would be in what we currently have as summer.

To make up for this time, we add an extra day to the month of February about every 4 yours. The rule is a little more complicated than that...

  • The year is evenly divisible by 4.
  • The year is NOT evenly divisible by 100...unless...
  • The year is evenly divisible by 400.

Like I said...it's complicated...

Atom Text Editor (2/28)

It is currently 3am and I couldn't sleep anymore so I am playing around on the web a little bit. In class we have been using BBEdit which is a great text editor but unfortunatly it is only available on Macs. I am a pc and linux guy so I am looking for a new program to use and I came across one called Atom. It is AWESOME! It has github integrated right into the editor so as you make changes to files and folders you can push them straight to a live branch. It also gives me a little more freedom for settings that are not super important but make writing code a little easier like auto wraping, custom tab spacing, and indent markers.

The best part is that it is available for Mac, PC, and Linux so I will have the same program availble to me on all of my computers (which is more rare than you would imagine). I actually started writing this post on my desktop PC then pushed it live. Then I switched over to my laptop that is running Kali linux where I pulled the updates and made some changes to the stylesheet. Then I finished editing this post on my MacBook. "Why?" you ask...because I can!
