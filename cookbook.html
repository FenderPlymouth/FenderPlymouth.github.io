Recipe Book Project Page

Create a web page with atleast 3 of your favorite recipes! The recipe book should have a table of contents linking to each recipe below, using internal links. Each recipe should have a table for the ingredients, a list of details, a list and/or paragraphs of the steps required, and a picture of what the food looks like. If you use recipes from the internet, be sure to add a link to the website.

Be sure to add links to add a navigation bar and links to all of your other pages!!!
