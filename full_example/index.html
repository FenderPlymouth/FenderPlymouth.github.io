Welcome to my example portfolio website! As we make our way through the year we will learn many things like how to create websites using HTML5, CSS, and Javascript. You will make use this knowledge to create your own personal websites to act as a portfolio to show off to the world your skills. As we move on past basic web design we will learn the programing language Python. With it, you will be able to develop games, apps, and tools. You will be able to post these projects to your website as well.

While there will be a few basic requirments to prove that you have mastered certain skills, it is my hope that the pages on this website and the programs yous post to your portfolio will be unique to you, giving you the opportunity to express your creativity and show off your new skills!

Just like any of the other projects we do in class, I encourage you to look at the source code and feel free to utilize any snippets you want. HOWEVER...you must include coments in your code that provides a direct link to where you got the code, and a detialed description of what the code does. If you can't explain the code in enough detail so Mrs. Fender understands what its function is, then you do not have a firm enough understanding to use it yourself.
